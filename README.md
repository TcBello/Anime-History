# Anime History
 
### About
Anime History is a Mobile App that you can browse animes and add them as your watched anime or completed anime.

### Features
- Create an account
- Browse animes
- Add or delete watched animes
- Upload profile picture
- Change password
- Change username

### Technology Used
- Flutter
- Node.js
- Express.js
- MongoDB
- Heroku
